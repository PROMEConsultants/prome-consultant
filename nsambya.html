Narrative Description of Project:

The entire project includes the following four components:

  • The Project Expressway, including major intersections with existing, ongoing, or future main roads and expressways within the Kampala Outer Beltway (KOB) Project area; this Expressway is expected to include a major bridge crossing at Ggaba.
  • The possible upgrading to service roads of some existing roads that will interact with the Project Expressway.
  • Radial links between the Project Expressway and the Northern Bypass.
  • Namugongo master plan and roads design.
